# Starcks × SportBet — Confronto offerte

Pagina di confronto tra le due proposte commerciali Starcks per SportBet:
**Offerta 01 — Starcks 360** (sponsorship esclusiva) e **Offerta 02 — Social** (costruzione asset social proprietari).

Pagina singola, autonoma, senza dipendenze esterne (immagini incluse nel file). Si apre in qualsiasi browser.

---

## Come pubblicarla online (GitHub Pages)

1. Crea un nuovo repository su GitHub (può essere **privato** o pubblico — la pagina ha comunque `noindex,nofollow`, quindi non finisce su Google).
2. Carica in questo repo i due file: `index.html` e `README.md` (basta trascinarli nella pagina "Add file → Upload files").
3. Vai su **Settings → Pages**.
4. Alla voce **Source** seleziona il branch `main` e la cartella `/ (root)`, poi **Save**.
5. Dopo 1–2 minuti GitHub genera il link pubblico, del tipo:
   `https://<tuo-utente>.github.io/<nome-repo>/`
6. Quel link è la pagina pronta da mandare a Nino.

> Nota: GitHub Pages funziona **anche con repo privati** (serve un account a pagamento per le pages private; su account free le pages sono pubbliche ma il link resta non indicizzato e non condiviso a meno che tu non lo dia).

---

## Alternativa senza pubblicare (apertura locale)

Se non vuoi metterla online, basta inviare il file `index.html` direttamente a Nino: lui fa doppio click e si apre nel browser. Funziona offline, tutto è dentro al file.
